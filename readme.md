# Swapi - React Native 
## Sanjeev Sharma

A react-native app that leverages the Swapi Star Wars API.

## Installation

1. Clone repo:  https://github.com/Codeofsanju/React-Native-Swapi.git
2. cd SwapiApp
3. npm i


## Usage

1. expo start
2. run local simulator or scan QRcode on Expo on mobile
